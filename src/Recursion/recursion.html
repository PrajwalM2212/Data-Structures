Recursive Steps

  1. Recursive problems are solved by reducing it to simpler problems of same kind until it can be solved directly
  2. The simplest problem is called the base case
  3. Base case stops recursion because it does not make another call to a method.

Structure of a program


        recursionMethod(arguments){
           if(base condition){
              // Handle the base case
           }else{
              // Handle the recursive case
              // Break the problem into simpler problems till it reaches base case
              recursionMethod(changed arguments)
           }
        }

A recursive method can have multiple base and recursive cases and each recursive call should take us closer to the base case
